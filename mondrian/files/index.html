Using RnMonitor OLAP Server API

This page explains how to perform Relational Online Analytical Processing (ROLAP) queries via Mondrian of files within the RnMonitor data warehouse.

The API is available at hhtp://localhost:8081/mondrian-rest

API Usage

There are currently four operations available on the API

  1. GET /getConnections
  2. GET /getSchema?connectionName=[name]
  3. GET /getMetadata?connectionName=[name]
  4. POST /query

To retrieve data from the data warehouse, we:

  1. Define the MDX query to produce the summarized data we want to visualize
  2. Submit the query using the mondrian-rest API end point hhtp://localhost:8081/mondrian-rest/query
  3. API returns a JSON Object containing the requested data
  4. Use JSON response to visualize or precess data on your own application

You can further experiment with submitting MDX queries to this dataset directly using the mondrian-rest query UI. MDX is an OLAP query syntax originally developed by Microsoft; Mondrian implements MDX for arbitrary Java/JDBC data sources. Because RnMonitor supplies a JDBC driver for RnMonitor datasets/files, we are able to use Mondrian to query those datasets/files using MDX syntax.
